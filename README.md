# 💼 Charlesson Portfolio

This project is a personal portfolio built to showcase my progress, projects, and skills as a Frontend Developer.

The goal is to create a modern, performant, and well-structured application following best practices in architecture, scalability, and user experience.

---

## 🚀 Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS

---

## 📁 Project Structure

The project follows a modular and scalable structure using the `src/` pattern:

* `app/` → Application routes and layout (Next.js App Router)
* `components/` → Reusable UI components
* `layout/` → Global layout elements (Header, Footer, etc.)
* `docs/` → Project documentation

## ⚙️ Getting Started

To run the project locally:

```bash
npm install
npm run dev
```

Then open:

```
http://localhost:3000
```
